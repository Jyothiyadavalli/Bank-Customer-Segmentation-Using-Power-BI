#  Bank Customer Segmentation using Power BI

##  Project Overview
This project focuses on **Bank Customer Segmentation** using **Power BI** to analyze customer demographics, transaction behavior, profitability, and risk levels.  
The objective is to convert raw banking data into **actionable business insights** that help banks improve customer retention, revenue growth, and risk management.

---

##  Business Problem
Banks generate a large amount of customer and transaction data, but without proper analysis it becomes difficult to:
- Identify **loyal and high-value customers**
- Detect **high-risk customers**
- Reduce customer churn
- Improve targeted marketing strategies

This project addresses these challenges using **RFM analysis and interactive Power BI dashboards**.

##  Key Insights
- Loyal customers contribute the **highest revenue**
- Transaction activity shows a **declining trend over time**
- Moderate-risk customers generate the **highest transaction value**
- High-risk customers require **continuous monitoring**

---

##  Recommendations
- Retain loyal customers through **rewards, personalized offers, and premium services**
- Re-engage dormant and lost customers using **targeted campaigns**
- Monitor high-revenue but high-risk customers closely to reduce financial risk
- Use age, location, and transaction patterns for **targeted marketing**
- Improve risk management by adjusting **credit policies for high-risk segments**

##  Conclusion
This Power BI dashboard provides a **comprehensive view of customer behavior, segmentation, and risk**, enabling banks to make **data-driven decisions** that improve profitability, customer retention, and financial stability.
